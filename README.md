Getting Started
1. Clone the repository
git clone https://github.com/your-username/emoji-meme-search.git
cd emoji-meme-search

2. Install dependencies
npm install

3. Run development server
npm run dev

4. Build for production
npm run build

# API Used
Imgflip Meme API
GET https://api.imgflip.com/get_memes


Returns meme name, URL, width, height, ID, etc.
